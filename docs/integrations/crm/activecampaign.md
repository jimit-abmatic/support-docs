# ActiveCampaign Integration

Connect Abmatic AI with ActiveCampaign to sync contacts and trigger automations.

## Overview

The ActiveCampaign integration enables:

- **Contact sync** - Sync contacts between systems
- **Field mapping** - Map custom fields
- **Automation triggers** - Trigger ActiveCampaign automations

## Connecting ActiveCampaign

1. Navigate to **Integrations** in the left sidebar
2. Find **ActiveCampaign** and click **Connect**
3. Enter your ActiveCampaign API URL and key
4. Click **Connect** to complete setup

## Sync Configuration

### What Syncs

| ActiveCampaign Object | Abmatic Object |
|-----------------------|----------------|
| Contacts | Contacts |
| Accounts | Accounts |

### Field Mapping

Map ActiveCampaign fields to Abmatic fields:

1. Go to integration settings
2. Click **Field Mapping**
3. Map fields between systems
4. Save mappings

## Importing from ActiveCampaign

Import contacts from ActiveCampaign:

1. Go to **Audiences** > **Contacts**
2. Click **Create New** > **From CRM**
3. Select **ActiveCampaign**
4. Choose a list to import
5. Review and import

## Pushing to ActiveCampaign

Push Abmatic contacts to ActiveCampaign:

1. Select contacts
2. Click **Push to CRM**
3. Select **ActiveCampaign**
4. Choose a list and confirm
