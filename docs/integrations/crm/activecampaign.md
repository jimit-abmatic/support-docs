# activecampaign Integration

Connect Abmatic AI with activecampaign for seamless data synchronization.

## Overview

The activecampaign integration enables:
- Two-way account and contact sync
- Field mapping customization
- Real-time data updates

## Setup

1. Go to **Integrations**
2. Click **activecampaign**
3. Click **Connect**
4. Authorize access

## Configuration

After connecting, configure:
- Sync direction
- Field mappings
- Sync frequency
