# Pipedrive Integration

Connect Abmatic AI with Pipedrive to sync accounts and contacts.

## Overview

The Pipedrive integration enables:

- **Two-way sync** - Sync organizations and persons
- **Field mapping** - Map custom fields between systems
- **Import from Pipedrive** - Import data from Pipedrive

## Connecting Pipedrive

1. Navigate to **Integrations** in the left sidebar
2. Find **Pipedrive** and click **Connect**
3. You'll be redirected to Pipedrive to authorize access
4. Grant Abmatic the requested permissions
5. You'll be returned to Abmatic with the connection active

## Sync Configuration

### What Syncs

| Pipedrive Object | Abmatic Object |
|------------------|----------------|
| Organizations | Accounts |
| Persons | Contacts |

### Field Mapping

Map Pipedrive fields to Abmatic fields:

1. Go to integration settings
2. Click **Field Mapping**
3. Map fields between systems
4. Save mappings

## Importing from Pipedrive

Import accounts or contacts from Pipedrive:

1. Go to **Audiences** > **Accounts** or **Contacts**
2. Click **Create New** > **From CRM**
3. Select **Pipedrive**
4. Choose data to import
5. Review and import

## Pushing to Pipedrive

Push Abmatic data to Pipedrive:

1. Select accounts or contacts
2. Click **Push to CRM**
3. Select **Pipedrive**
4. Map fields and confirm
