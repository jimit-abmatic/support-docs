# Outreach Integration

Connect Abmatic AI with Outreach to sync contacts and track engagement.

## Overview

The Outreach integration enables:

- **Contact sync** - Sync contacts between systems
- **Sequence integration** - Align with Outreach sequences
- **Activity tracking** - Track Outreach engagement in Abmatic

## Connecting Outreach

1. Navigate to **Integrations** in the left sidebar
2. Find **Outreach** and click **Connect**
3. Authorize access to your Outreach account
4. Complete the connection setup

## Sync Configuration

### What Syncs

| Outreach Object | Abmatic Object |
|-----------------|----------------|
| Prospects | Contacts |
| Accounts | Accounts |

### Field Mapping

Map Outreach fields to Abmatic fields:

1. Go to integration settings
2. Click **Field Mapping**
3. Map fields between systems
4. Save mappings

## Importing from Outreach

Import contacts from Outreach:

1. Go to **Audiences** > **Contacts**
2. Click **Create New** > **From CRM**
3. Select **Outreach**
4. Choose data to import
5. Review and import

## Pushing to Outreach

Push Abmatic contacts to Outreach:

1. Select contacts
2. Click **Push to CRM**
3. Select **Outreach**
4. Map fields and confirm
