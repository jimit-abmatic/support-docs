## Key Metrics

### Engagement Metrics

- **Page Views** - Pages viewed by target accounts
- **Time on Site** - Average session duration
- **Return Visits** - Repeat visitor frequency

### Campaign Metrics

- **Impressions** - How many saw your personalization
- **Conversions** - Goals completed
- **Conversion Rate** - Percentage who converted

### Account Metrics

- **Engaged Accounts** - Accounts showing activity
- **New Accounts** - Newly identified companies
- **Account Score** - Overall engagement score

## Using Analytics

### Navigate to Analytics

1. Click **Analytics** in the left sidebar
2. Choose **Reports** or **Dashboards**

### Create a Report

1. Click **Create Report**
2. Select metrics and dimensions
3. Choose date range
4. Apply filters
5. Save and share

### Build a Dashboard

1. Click **Create Dashboard**
2. Add widgets (charts, tables, KPIs)
3. Configure each widget
4. Arrange layout
5. Save

## Best Practices

- **Review regularly** - Check analytics weekly or daily
- **Compare periods** - Use date comparisons to spot trends
- **Segment data** - Analyze by audience, campaign, or channel
- **Share insights** - Keep stakeholders informed
